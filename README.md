# One Identity Manager - Zero to Hero

## 📚 Complete Learning Guide for OneIM

A comprehensive, professionally-structured learning platform covering Identity Governance & Administration (IGA) concepts, OneIM platform architecture, installation, configuration, and advanced customization.

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start learning from the Introduction section

- **IGA Fundamentals** - Learn identity governance concepts

- **Installation** - Step-by-step setup guide
